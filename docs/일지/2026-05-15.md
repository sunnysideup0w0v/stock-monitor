이전 세션에서 작업을 시작했으나 컨텍스트가 종료되어 이번 세션에서 마무리했다.

#### 구현 내용

**`RealtimeQuoteManager.swift` (신규, actor)**

KIS WebSocket 연결의 전체 생명주기를 담당한다.

- **승인키 발급**: `POST /oauth2/Approval` — REST 토큰과 별개로 WebSocket 전용 승인키가 필요하다. 필드명이 `secretkey` (REST는 `appsecret`)로 다른 점에 주의.
- **WebSocket 연결**: 실전 `ws://ops.koreainvestment.com:21000` / 모의 `:31000`
- **구독 메시지**: `tr_id: "H0STCNT0"`, `tr_type: "1"(구독) / "2"(해제)`
- **메시지 파싱**: `{encrypt}|{tr_id}|{count}|{data}` 형식. `^` 구분자로 필드 분리.
  - `fields[0]` 종목코드, `[2]` 현재가, `[3]` 부호, `[4]` 변동폭, `[5]` 등락률, `[13]` 거래량
  - 암호화 메시지(`parts[0] == "1"`)는 현재 무시
- **지수 백오프 재연결**: 1s → 2s → 4s → … → 60s 상한
- **20초 핑**: `webSocketTask?.sendPing { _ in }` 으로 연결 유지

**`QuoteManager.swift` 변경**

```swift
func startRealtime(credentials: BrokerCredentials, isMock: Bool)
func stopRealtime()
func updateFromRealtime(quote: StockQuote)   // WebSocket 수신 시 MainActor에서 호출
```

WebSocket과 폴링이 공존하는 구조다. WebSocket이 연결되면 실시간 업데이트가 오고, 폴링도 별도로 유지되어 WebSocket 미연결 상태의 폴백 역할을 한다.

**`Info.plist` 변경**

KIS WebSocket이 `ws://`(비TLS)를 사용하므로 ATS 예외 도메인을 추가했다.

```xml
<key>NSExceptionDomains</key>
<dict>
    <key>ops.koreainvestment.com</key>
    <dict>
        <key>NSExceptionAllowsInsecureHTTPLoads</key>
        <true/>
    </dict>
</dict>
```

### 설정 UI 리팩토링 — 공통 컴포넌트 추출

**배경**: 계좌 연결 탭에만 적용되어 있던 좌우 패딩이 다른 탭(관심종목·포트폴리오·알림설정)에는 없어 일관성이 깨져 있었다.

#### SwiftUI 컴포넌트화 — React와의 비교

React의 `children` prop과 동일한 개념이 SwiftUI에서는 `@ViewBuilder`다.

```swift
// SwiftUI
struct SettingsTabContainer<Content: View>: View {
    let title: String
    let content: Content

    init(title: String, @ViewBuilder content: () -> Content) {
        self.title = title
        self.content = content()
    }

    var body: some View {
        VStack(alignment: .leading, spacing: 12) {
            Text(title).font(.title2).bold()
            content
        }
        .padding([.horizontal, .bottom], 8)
    }
}
```

`<Content: View>` 제네릭이 TypeScript의 `ReactNode` 타입 제약에 해당하고, `@ViewBuilder`가 클로저 안의 여러 뷰를 하나의 `Content`로 합쳐준다.

#### 추출한 컴포넌트

| 컴포넌트 | 역할 |
|----------|------|
| `SettingsTabContainer` | 타이틀 + 좌우하단 8px 패딩. 모든 설정 탭에 적용 |
| `SettingsFormSection` | Divider + `.headline` 섹션 제목. 종목 추가 폼 3곳에서 공유 |

## 기술적 결정 사항

| 결정 | 이유 |
|------|------|
| WebSocket + 폴링 병행 | WebSocket은 장 시간에만 동작. 폴링이 폴백 역할을 하면 Mock 모드나 장 외 시간에도 시세 갱신이 가능 |
| 암호화 메시지 스킵 | KIS WebSocket은 일부 메시지를 AES 암호화해서 전송. 현재는 미암호화(`parts[0] == "0"`) 메시지만 처리하고 암호화는 추후 대응 |
